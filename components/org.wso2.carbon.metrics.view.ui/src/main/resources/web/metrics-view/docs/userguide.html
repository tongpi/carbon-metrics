指标视图

介绍

您可以使用“指标视图”查看不同图表中的JVM级别和应用程序级别指标。

重新加载指标

您可以使用重新加载按钮来生成所有图表，并且您可以选择视图，来源和时间范围。

默认情况下，会选择多个指标视图，并使用最近5分钟的数据生成图表。

查看

所有指标图表都分组到不同的视图中。这意味着每个视图可以有一个或多个图表。 在重新加载所有图表之前，您必须至少选择一个视图。

资源

来源下拉菜单显示了一系列指标来源。此下拉列表中显示的值是计算机的主机名，它向配置的数据库报告指标。即使计算机主机名是Metrics记者的默认源，用户也可以在度量标准配置中为源指定任何有意义的名称。在这种情况下，配置的源名称将显示在此下拉列表中。

时间范围

这是另一个下拉菜单，用于选择相对时间范围并设置自定义时间范围。当您从下拉列表中选择自定义时，您会看到两个输入以输入自定义时间范围。

公制图表

度量标准图表是多行图表,每个图表中都会显示图例。您可以切换图表上方显示的复选框来隐藏或查看图表行。您必须至少选择一个复选框。

X轴表示度量数据的时间戳，Y轴线取决于每种图表类型。
